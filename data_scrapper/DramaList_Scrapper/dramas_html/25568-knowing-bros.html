It is a sitcom talk show program set in a school. In each episode, new celebrity guests appear as transfer students at the 'Brother School' where seven mischievous brother students wait for them. The show consists of several dynamic sessions. In ‘Guess About Me,’ Celebrity guests create random, secret questions about themselves. Then the hosts called 'Knowledgeable Bros' make humorous guesses to get the answers. Another session is ‘Bros Inside’, an ad-lib skit show, where entertainers are given only the setting for a story and have to perform completely ad-libbed skits. (Source: JTBC) Edit Translation
